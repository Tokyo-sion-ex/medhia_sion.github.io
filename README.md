**【説明】**

YouTubeのリンクを特定の場所に貼り、タイトルと説明を自由に記入してできます。
通常の動画はもちろん、**ライブ動画**や**長編動画**も対応！
htmlだけで動作しているので、通常より軽く動作(してくれる筈...!)します。




**【内部アイコンについて】**

アイコンはYouTubeのアイコンを利用させていただいておりますが、アイコンについては別途ライセンスを公開しておきますので、商用利用する際は必ずご確認下さい。
また、現在CC0として作品を公開していますが、あくまでも**コードのみ**ということにしています。




**【操作方法】**

「ホーム」⋯ 動画を再生・選択ができます。

「動画追加」⋯ 動画を追加できます。

「履歴」⋯ 動画を追加した日時が見れます。(※デフォルトで追加されている場合は"NAN"と書かれています)

「?ボタン」⋯ 操作方法が書かれています。

「☀ / ☽ ボタン」⋯ ライトモード / ダークモードを切り替えられます。

「✕ or 削除 ボタン」⋯ 動画を削除できます(※操作の取り消しはできません)

「⚙️ ボタン」⋯ テーマカラーを変えられます。
